Valgrind permet de deceler les fuites de memoires et plein d'autres choses (conditionnal jump etc...).
Par défaut il detecte les allocations faite par des bibliothèques tierces comme une fuite.

Pour éviter celà pour la bibliothèque SDL (pas encore pour FMOD), il faut ajouter un fichier de suppression qui se trouve dans le repertoire courant : "valgrind_lif7.supp".

Executer cette commande pour effectuer les tests de memoires. Les erreurs seront repertoriés dans un fichier error.txt :

valgrind --leak-check=full --num-callers=50 --suppressions=./valgrind.supp --show-reachable=yes -v --log-file=error.txt bin/blockade

Pour les autres paramètres, RTFM...
Doc officiel : http://valgrind.org/docs/
